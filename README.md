# Getting Started with Create React App

I am currently working on developing a user authentication system that comprises a sign-up module, login module, and profile management module. The sign-up module will enable users to create an account by providing their personal information, while the login module will authenticate users using their usernames and password. Additionally, the profile management module will allow users to view or edit their profile information after successful login.

Technologies to be used: React JS, Node JS, Express JS, MongoDB, JavaScript, Bootstrap 5, CSS3, & HTML5
